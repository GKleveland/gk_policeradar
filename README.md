# gk_policeradar
Nummerpladescanner Til Politiet

# Installation
- Drag And Drop ind i resource folder.
- I server.cfg indsæt: "start gk_policeradar"
- Værsgo ;)

# Features
- Scanner det nærmeste køretøj foran og bagved.
- Slå til/fra på numpad5
- Fastlås måling på numpad8
